Theme Customization

Match the map to your brand — recolor the location labels and the built-in control buttons through plain attributes. No CSS or build step required.

Open the demo →

When to use this pattern

Use the theme attributes when the default look doesn't fit the surrounding page. You can set the label font, label text/background color, and the control buttons' foreground/background in both their normal and active states. Anything you leave unset falls back to the built-in defaults.

Step-by-step setup

1. Theme the labels

Labels are drawn on the canvas. Set their text color, background "pill" color, and optionally the font family. 

<wayfinder-map
  data-url="../datas/SGC_v001.json"
  default-floor="L3"
  map-label-font-color="#1d3557"
  map-label-background-color="#f1faee"
  map-label-font-family="'Inter', system-ui, sans-serif"
></wayfinder-map>

2. Theme the controls

The level selector and search buttons take foreground (icon/text) and background colors, with separate values for the active (selected) state. 

<wayfinder-map
  ...
  level-selector
  search-control
  control-fg-color="#1d3557"
  control-bg-color="#ffffff"
  control-active-fg-color="#ffffff"
  control-active-bg-color="#e63946"
></wayfinder-map>

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1,
          maximum-scale=1, user-scalable=no" />
    <title>Saigon Centre Map</title>
    <style>
      html, body { margin: 0; height: 100%; overflow: hidden; }
      wayfinder-map { display: block; width: 100vw; height: 100vh; }
    </style>
  </head>
  <body>
    <wayfinder-map
      data-url="../datas/SGC_v001.json"
      default-floor="L3"
      level-selector
      search-control
      enable-rotation
      map-label-font-color="#1d3557"
      map-label-background-color="#f1faee"
      control-fg-color="#1d3557"
      control-bg-color="#ffffff"
      control-active-fg-color="#ffffff"
      control-active-bg-color="#e63946"
    ></wayfinder-map>

    <script type="module">
      import "../dist/wayfinder-map.esm.js";
      const map = document.querySelector("wayfinder-map");
      map.addEventListener("error", (e) => console.error("Map error:", e.detail?.error || e));
    </script>
  </body>
</html>
