Bare Canvas Integration

The simplest way to embed the Saigon Centre map — a full interactive canvas with pan, zoom, rotation, and labels, but no built-in UI controls.

Open the demo →

When to use this pattern

Use the bare canvas when you want to build your own UI on top of the map, or when you only need the interactive map without any controls. You get a pannable, zoomable canvas with location labels rendered automatically. You will not get a floor selector or search panel — for those, see the Default Controls guide.

Step-by-step setup

1. Include the script

Load the ES module bundle. It registers the <wayfinder-map> custom element automatically. Everything it needs (rbush included) is bundled, so no import map is required. 

<script type="module">
  import "../dist/wayfinder-map.esm.js";
</script>

2. Add the map element

Place a <wayfinder-map> element in your HTML. The only required attribute is data-url — a single, self-contained bundle that carries geometry, the destination catalog, the navmesh, and transitions. There is no separate map-url. 

<wayfinder-map
  data-url="../datas/SGC_v001.json"
  default-floor="L3"
></wayfinder-map>

3. (Optional) Responsive rendering

The component detects viewport width at a 768px breakpoint. Attributes prefixed with desktop- apply above 768px and mobile- below. The Saigon Centre bundle renders in raw coordinates (render-scale defaults to 1), so these are usually unnecessary — tune only if you need per-device zoom or label sizing. 

<wayfinder-map
  data-url="../datas/SGC_v001.json"
  default-floor="L3"
  desktop-max-zoom="3"   mobile-max-zoom="2"
  desktop-min-zoom="fit" mobile-min-zoom="fit"
  desktop-label-font-size="6" mobile-label-font-size="4"
></wayfinder-map>

4. Handle errors

Listen for the error event to catch data-loading or rendering failures. 

const map = document.querySelector("wayfinder-map");
map.addEventListener("error", (event) => {
  console.error("Map error:", event.detail?.error || event);
});

Full source code

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1,
          maximum-scale=1, user-scalable=no" />
    <title>Saigon Centre Map</title>
    <style>
      html, body { margin: 0; height: 100%; overflow: hidden; }
      wayfinder-map { display: block; width: 100vw; height: 100vh; }
    </style>
  </head>
  <body>
    <wayfinder-map
      data-url="../datas/SGC_v001.json"
      default-floor="L3"
    ></wayfinder-map>

    <script type="module">
      import "../dist/wayfinder-map.esm.js";
      const map = document.querySelector("wayfinder-map");
      map.addEventListener("error", (e) => console.error("Map error:", e.detail?.error || e));
    </script>
  </body>
</html>

Attribute reference

AttributeDescription
data-urlRequired. URL of the self-contained map bundle.
default-floorFloor code to show first (e.g. L3). Codes: B2, B1, L1, L2, L3.
render-scaleWorld-coordinate scale. Defaults to 1 (raw CMS coordinates).
enable-rotationAllow two-finger / gesture rotation of the canvas.
