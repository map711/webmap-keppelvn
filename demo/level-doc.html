Level Selector Only

The bare canvas plus the built-in level selector — nothing else. Visitors can pan, zoom, read labels, and switch between the five floors, but there is no search panel and no rotation. Use this when you want floor switching without the rest of the default chrome.

Open the demo →

When to use this pattern

Reach for the level-selector-only setup when your embed needs nothing more than floor switching — for example a directory board or a kiosk where search is handled elsewhere (or not at all) and you want the canvas locked to its default orientation. It is the default controls pattern with search-control and enable-rotation omitted.

Step-by-step setup

1. Include the script

Same as the bare canvas: a single module import. 

<script type="module">
  import "../dist/wayfinder-map.esm.js";
</script>

2. Enable just the level selector

Add the single boolean attribute level-selector (floor switcher). Leave off search-control and enable-rotation — the controls are independent, so omitting them simply hides the search panel and locks rotation. 

<wayfinder-map
  maps-url="../datas/maps_SGC_v001.json.gz"
  datas-url="../datas/datas_SGC_v001.json.gz"
  default-floor="L3"
  level-selector
></wayfinder-map>

3. (Optional) Listen for browse events

The component re-emits engine events as DOM events you can hook into. 

const map = document.querySelector("wayfinder-map");
map.addEventListener("data-loaded", () => console.log("bundle ready"));
map.addEventListener("floor-changed", (e) => console.log("floor:", e.detail));
map.addEventListener("location-tap", (e) => console.log("tapped:", e.detail));
map.addEventListener("location-disambiguate", (e) => console.log("multi-tenant:", e.detail));

    <script type="module">
      import "../dist/wayfinder-map.esm.js";
      const map = document.querySelector("wayfinder-map");
      map.addEventListener("error", (e) => console.error("Map error:", e.detail?.error || e));
    </script>
  </body>
</html>

Attribute reference

AttributeDescription
maps-urlRequired. URL of the maps_… half (geometry + navmesh + transitions).
datas-urlRequired. URL of the datas_… half (shop directory + categories).
default-floorFloor code to show first (B2, B1, L1, L2, L3).
level-selectorBoolean. Show the built-in floor switcher.
