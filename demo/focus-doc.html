Demos / Focus a Shop Guide

Focus a Shop at Startup

Open the map already zoomed onto a specific shop — the component switches to the shop's floor, animates a zoom-in, and drops an end pin. One declarative attribute: focus-shop-id.

Open the demo →

When to use this pattern

Use focus-shop-id when you deep-link into the map from somewhere else — a directory listing, a QR code on a shopfront, a "show on map" button. The visitor lands directly on the destination instead of the whole-floor overview. Clearing the focus (the control's close / back action) returns to browse mode.

Note: this is distinct from the carried-over focus-node-id attribute, which targets a navmesh graph node and belongs to Phase-2 routing. The Saigon Centre bundle ships a per-level navmesh rather than a flat node graph, so for Phase-1 browse you focus a shop by id. focus-shop-id is the declarative front door to the engine's focusLocation('shop:<id>') path.

Step-by-step setup

1. Find the shop id

Shop ids come from the bundle's shops[] array. For example, shop 10 is Starbucks (unit 108, floor L3).

2. Set the attribute

Add focus-shop-id to the element. The focus is applied once the bundle loads — if the shop is on another floor, the map switches floors first. Pair it with the default controls so the visitor can clear the focus and keep browsing. 

<wayfinder-map
  maps-url="../datas/maps_SGC_v001.json.gz"
  datas-url="../datas/datas_SGC_v001.json.gz"
  default-floor="L3"
  level-selector
  search-control
  enable-rotation
  focus-shop-id="10"
></wayfinder-map>

3. (Optional) Focus from script instead

The attribute is sugar over the public method. To focus reactively (e.g. from your own button), call focusLocation directly. You can also change focus-shop-id at runtime and the map re-focuses. 

const map = document.querySelector("wayfinder-map");
map.addEventListener("data-loaded", () => {
  map.focusLocation("shop:10");        // same as focus-shop-id="10"
});

Attribute reference

AttributeDescription
focus-shop-idShop id to focus at startup; switches floor, zooms in, drops a pin. Re-applies if changed at runtime.
focus-node-idPhase-2 navmesh-node focus (no effect on the current browse bundle).
level-selector / search-controlRecommended alongside focus so the visitor can clear it and keep browsing.
